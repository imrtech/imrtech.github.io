Project Overview

Demographic information including bacteria collected from volunteer's belly buttons was collected. The data on this dashboard will display each volunteer's information and belly button biodiversity.

The drop down list below labeled Test Subject ID can be used to select information from the different volunteers. As you select an ID, the charts below "Top 10 Bateria Cultures Found", "Belly Button Washing Frequency Scrubs per week" and "Bacteria Cultures Per Sample" will change according to your selection.
